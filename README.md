# 📊 Sales Analysis System (Using SQL, Pandas, NumPy)

---

## 1. 📌 Project Overview
The **Sales Analysis System** is a data-driven project designed to analyze sales data and generate meaningful insights using:
* **SQL (via SQLite)** → Data storage & querying
* **Pandas** → Data manipulation & analysis
* **NumPy** → Numerical computations
* **Matplotlib & Seaborn** → Data visualizations

🎯 **Goal:** Help businesses understand sales trends, customer behavior, and performance metrics to make better decisions.

---

## 2. 📂 Project Structure
* `generate_data.py`: Python script to generate random dummy data for the project (`customers.csv`, `products.csv`, `sales.csv`).
* `sales_analysis.py`: Main Python script that sets up the database, performs SQL queries, computes KPIs using Pandas/NumPy, and exports data visualizations.
* `app.py`: Streamlit frontend application to display the interactive web dashboard.
* `requirements.txt`: List of Python packages required to run this project.
* `customers.csv`, `products.csv`, `sales.csv`: Sample datasets generated by `generate_data.py`.
* `dashboard_visuals/`: Directory where generated charts and graphs are saved.
* `sales_system.db`: SQLite database file generated during execution.

---

## 3. 🚀 How to Run the Project

### Step 1: Install Dependencies
Open your terminal and install the required Python packages:
```bash
pip install -r requirements.txt
```

### Step 2: Generate Sample Data (If not already generated)
If the `.csv` files are missing, run the following command to create them:
```bash
python generate_data.py
```

### Step 3: Run the Analysis
Execute the main analysis script to see insights printed to the terminal and charts saved in the `dashboard_visuals` folder:
```bash
python sales_analysis.py
```

### Step 4: Run the Web Dashboard
Launch the interactive Streamlit frontend application:
```bash
python -m streamlit run app.py
```
This will open the web dashboard in your default browser at `http://localhost:8501`.

---

## 4. 📊 Output Generated

### Terminal Output
The script prints the following insights directly to your console:
- SQL Query Results: Total Revenue, Top 5 Products
- Pandas Analysis: Total Revenue, Monthly Sales Trends, Customer Segmentation
- NumPy Calculations: Monthly Growth Rate Percentage

### Visualizations (`dashboard_visuals/`)
The script generates `.png` files for reports:
1. `monthly_sales_trend.png`: A line chart depicting revenue fluctuations across the year.
2. `top_10_products.png`: A bar chart showing the best-selling products.
3. `category_distribution.png`: A pie chart illustrating the revenue share across different product categories.
